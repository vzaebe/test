10. Справочник статусов

Полный перечень из Приложений 3, 4, 5 руководства пользователя АССКД. Все три категории используют общий компонент .st-row (см. раздел 1 — Дизайн-система). Цвет = смысл; форма строки одинакова по всему модулю.

Приложения 3–5 · Унифицировано
Каждый статус ниже = пара «индикатор .si + бейдж .bg». Один цвет — один смысл во всём модуле. См. раздел 1. Дизайн-система → Статусные индикаторы.
